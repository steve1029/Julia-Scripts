파이썬의 mpi4py package 는 openmpi 기반으로 만들어져 있다. 따라서 .py 를 mpi로 실행하고 싶다면


반드시 mpiexec.openmpi 명령어를 이용해서 실행해야하므로 구별해서 사용하자. 안그럼 에러가 날 수 있다.


ldg@yhome3:~/pyscript/MPI$ mpiexec.openmpi -host y101,y102,y103,y104 python broadcast.py


이런식으로 broadcast.py 파일을 host 를 지정해주어서 실행할 수 있다.

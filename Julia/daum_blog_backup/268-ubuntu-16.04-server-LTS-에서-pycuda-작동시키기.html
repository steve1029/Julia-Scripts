ubuntu 16.04 server LTS 에서 pycuda 작동시키기

리눅스/우분투 cuda 환경 구축

2019-03-11 15:24:18

1. 우분투 서버 버전을 설치한다. 반드시 메인보드에서 secure boot 를 끄고, '기타 OS' 로 설정한 뒤 Above 4G Decoding 옵션이 지원될 경우, Enable으로 활성화한다.

2. 인터넷을 연결한다.

3. SSH 를 연결한다.

4. 다음 링크의 순서를 따라 nvidia driver 를 설치한다.
https://hiseon.me/2018/02/17/install_nvidia_driver/
나는 190311 기준으로 GTX TITAN Black 을 위해 nvidia-418 을 설치하였다.

5. pycuda documentation 에서 ubuntu 설치 가이드를 가보면 매우 간단한 설치 방법이 적혀있다.

# apt install nvidia-cuda-toolkit nvidia-cuda-dev python3-pycuda python3-numpy

6. 공식 홈피 대문에 있는 예제코드를 python3 형식으로 바꾼 뒤 돌려보면 끝.

리눅스에 일단 anaconda python 2 나 python 3 를 설치한다.

그 다음 conda 를 이용해서 다른 파이썬을 설치하는 것.

conda 는 virtualenv 라고 하는 가상 환경인거 같다. virtualenv 는 무거운데, conda 는 그 중에서 가벼운 편에 속함.

anaconda official documentation 에서 두 개를 다 설치했으면 다음의 명령으로 왔다갔다 할 수 있다

# source activate py27
# source activate py36

사용을 마치고 원래대로 돌아오려면

# source deactivate py27


근데 클러스터에서는 쓸 수 없다. 가상 환경을 만들어서 python 2, 3 을 동시에 쓰는 방법인데, 리눅스에서는


가상 파일 시스템으로 /proc, /dev/pts 등등이 있다. 두 가지 파이썬을 사용하게 되면 이 두 가상 파일이 항상 사용 중이 되므로


umount 를 할 수 없어지고, 강제적으로 umount 할 경우 모든 노드의 /proc 이 강제로 umount 되는 현상이 발생한다. 매우 귀찮다.

Anaconda 를 설치하면 파이썬과 주피터 노트북, 랩이 같이 설치된다.

이제 juliapro 를 설치한다.

juliapro 안에는 julia 특화용 atom 이 같이 설치되어 있다. 이를 실행한다.

julia terminal 에서 다음을 실행한다.

using Pkg
Pkg.add("IJulia")

이제 anaconda manager 에서 jupyter 노트북이나 랩을 실행하면 julia kernel 을 사용할 수 있다.

통합 개발툴은 python은 spyder, 줄리아는 줄리아pro의 atom 을 이용하도록 하자.

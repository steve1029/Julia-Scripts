sys 모듈

파이썬 인터프리터가 제공하는 변수들과 함수들을 직접 제어할 수 있게 해주는 모듈.

여기서 말하는 sys 는 system 의 약자이며, 컴퓨터에 설치된 os 를 말하는 것이 아니라 파이썬을 실행하고 있는 interpreter 를 system 이라고

칭하는 것이다.


1. sys.exit()

해당 스크립트를 강제로 종료한다.

쉘에서 ctrl + z 나 ctrl + d 로 프로세스를 종료하는 것과 같은 기능이다.

2. sys.path

파이썬 모듈들이 저장되어 있는 위치를 반환한다. 즉 이 interpreter 가 설치된 경로, 그리고 참조하는 위치를 return 하는 함수이다.


이 위치에 있는 모듈은 경로에 상관없이 어디에서나 불러올 수 있다.

내가 만든 모듈은 항상 경로를 지정해줘야 하지만 여기에 저장되어 있다면 그럴 필요가 없는 것.

>>> sys.path.append("C:/Python/MyModules")

라고 해주면 sys.path 에 새로운 경로가 추가되고, 이 경로안에 있는 모듈들을 불러와서 항상 쓸수가 있다.


하지만 영구히 추가되는 것이 아니고 해당 스크립트에서만 추가된 것이다.

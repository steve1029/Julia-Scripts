locale 에러가 날 시

리눅스/시행 착오 노트

2017-07-21 16:00:57

warning: Please check that your locale settings:
LANGUAGE = "en_US:en"
LC_ALL = (unset)
LANG = "en_US.UTF-8"

이런 에러가 날 수 있다. 주로 debootstrap 을 이용해 받은 노드의 루트파일시스템에 chroot 로 들어가면 발생하는데,
해결 방법은 다음과 같다.

# aptitude install -y locales
# locale-gen "en_US.UTF-8"

이러면 새로운 프로그램이 열릴 것이며, en_US.UTF-8 을 찾아서 설치하도록한다. 모든 설정을 다 설치할 필요는 없다.


이 방법이 더 좋다

# ㅣocale-gen "en_US.UTF-8"

# dpkg-reconfigure locales

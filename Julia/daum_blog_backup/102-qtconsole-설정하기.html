여기가 ipython 공식 Documentation 사이트이다. 여기서 본 내용을 간단히 정리한다.


윈도우의 경우 qtconsole 의 설정파일은 다음 위치에 있다.

C:\Documents and Settings\컴퓨터이름\.ipython\profile_default


혹은 powershell 에서 다음을 입력하면 위치를 알려준다.


>>> ipython locate

하지만 처음에는 아무 config 파일이 없으므로 만들어줘야 한다.


powershell을 열고 다음 명령을 입력하자.

자신의 파이썬 버전에 따라 둘 중 아무거나 입력하면 된다.

>>>ipython2 profile create [profilename]

>>> ipython3 profile create [profilename]


[profilename] 이 부분을 blank 로 두면 default 값으로 파일이 생성된다. 그러니 아무것도 입력 안하는게 제일 속편하다.

그러면 위의 디렉토리에 config 파일이 형성될 것이다. 


공식 홈페이지에서 제공하는 example 은 다음과 같다.

# sample ipython_config.py
c = get_config()

c.TerminalIPythonApp.display_banner = True
c.InteractiveShellApp.log_level = 20
c.InteractiveShellApp.extensions = [
    'myextension'
]
c.InteractiveShellApp.exec_lines = [
    'import numpy',
    'import scipy'
]
c.InteractiveShellApp.exec_files = [
    'mycode.py',
    'fancy.ipy'
]
c.InteractiveShell.autoindent = True
c.InteractiveShell.colors = 'LightBG'
c.InteractiveShell.confirm_exit = False
c.InteractiveShell.editor = 'nano'
c.InteractiveShell.xmode = 'Context'

c.PrefilterManager.multi_line_specials = True

c.AliasManager.user_aliases = [
 ('la', 'ls -al')
]



각 항목에 대한 설명은 다음 사이트에 잘 나와 있다. 


http://ipython.readthedocs.io/en/stable/config/options/index.html


Terminal IPython options 은 ipython_config.py 에 적용되는 항목이다.

IPython kernel options 은 ipython_kernel_config.py 에 적용되는 항목이다.

ipython kernel 에는 ipython_config.py 에 적용된 설정 또한 적용된다.


그리고 ipython_config.py 파일 내에도 설명이 잘 나와 있으니 읽어보고 바꾸면 된다.

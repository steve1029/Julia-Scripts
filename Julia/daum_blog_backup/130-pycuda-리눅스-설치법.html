pycuda 리눅스 설치법

컴퓨터 언어/CUDA

2017-01-23 10:42:22

aptitude 를 이용하여 설치하면 옛날버전의 nvidia toolkit 이 적용되므로 쓸 수 없다.


먼저 nvidia toolkit 을 공식홈페이지에서 다운 받는다.


여기서 리눅스 버전을 다운 받게 되면 데비안은 없고 우분투만 있다. 난 데비안 stable 을 쓰므로,


우분투의 14.04 버전을 다운받아야 한다. 물론 네트워크 버전을 받도록 하자.


그리고 xftp 로 파일을 옮긴 뒤 다음의 과정을 따라하자.


# dpkg -i [filename]

# aptitude update

# aptitude install -y cuda


이러면 끝난다. 편리하고 좋다. 여기서 dpkg 는 aptitude 의 앱 src 에 엔비디아의 사이트를 참조하도록 추가하는 명령인듯 보인다.


그리고 conda 를 이용해 pip 을 설치한다.


# conda install pip

# pip install pycuda


conda install 을 하면 아나콘다에서 제공하는 파이썬 패키지를 받는 것. 하지만 pycuda 는 아나콘다에서 제공하지 않는다.


pip 은 파이썬 공식 홈페이지에서 가져오는 듯. 그래서 웬만한건 다 있다. 


마지막으로 하나 박아주자.


# pip install reikna


기환형이 코멘트를 정리해주셨다.

1. ubuntu에서 deb 패키지를 제공할 때

  # apt-get install reikna


2. ubuntu에서 deb 패키지가 없을 때

  # pip install reikna


3. anaconda가 패키지를 제공할 때

  $ conda install reikna


4. anaconda가 패키지가 없을 때

  $ conda install pip

  $ pip install reikna


1번은 리눅스 시스템에 설치되고,

2~4번은 anaconda/ 디렉토리 밑에 설치되는 것이 차이




aptitude를 이용하여 python-pycuda 를 설치하면 nvidia toolkit 이 자동으로 같이 설치된다.

cuda toolkit 과 cuda compiler 가 제대로 설치되었는지 확인하기 위해서 다음 명령을 입력해보자

# nvcc --version

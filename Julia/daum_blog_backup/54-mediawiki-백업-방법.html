mediawiki 백업 방법

리눅스/서버 앱 설정 방법

2016-09-28 16:23:38

# aptitude install -y automysqlbackup

이 앱은 내 mysql 을 자동으로 백업해주는 놈이다.

모든 백업 데이터는 /var/lib/automysqlbackup/ 에 저장된다.

여기 보면 daily, weekly, monthly 로 나오는데, 자동으로 매일, 매주, 매월 해주나보다.

수동으로 백업하려면 다음을 입력하면 된다.

# automysqlbackup

그리고 백업 파일로 데이터베이스를 복구하는 방법은 다음과 같다.

# gunzip < /var/lib/automysqlbackup/weekly/nolwiki/nolwiki_2016-09-28_16h10m.sql.gz|mysql -uUSER -pPASSWORD nolwiki

여기에서 USER, PASSWORD 자리에 사용자의 아이디와 비번을 넣으면 된다.

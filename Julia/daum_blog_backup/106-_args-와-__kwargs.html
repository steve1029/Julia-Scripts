*args 와 **kwargs

컴퓨터 언어/Python

2016-11-10 20:00:39

함수를 정의할 때 사용할 argument 들을 일일이 지정해줄 수도 있지만,


argument 의 개수가 몇 개인지 정확히 알 수 없는 경우는 어떻게 쓸까?


그 때 필요한 것이 *args 와 **kwargs 이다.


args : arguments

kwargs : keyword arguments


다음의 예제를 보자.


def required_arg(*args, **kwargs):


"""

parameters

------------------------

I : float, int

magnitude of incident E field


theta, incidentangle : float

incident angle

-------------------------

these parameters can be key of kwargs


"""


# parameters

mum = 1e-6

nm  = 1e-9

I   = 1

incidentangle = 0

wavelength = 700 * nm



print kwargs.keys()

print kwargs.values()


# change default value of parameters according to kwargs

for i, key in enumerate(kwargs.keys()):

if key == 'I':

I = kwargs['I']


if key == 'theta' or key == 'incidentangle':

if key == 'incidentangle' :

incidentangle = kwargs['incidentangle']

else :

incidentangle = kwargs['theta']


print 'intensity : ', I

print 'incident angle : ', incidentangle

print 'wavelength :', wavelength


import numpy as np

required_arg()

required_arg(I=2)

required_arg(I=2,incidentangle=np.pi/2)

required_arg(I=2,theta=np.pi/2)



*args 나 **kwargs 둘 중 하나만 사용해도 되고 둘 다 사용해도 된다. 단, 둘 다 사용할 땐 반드시 **kwargs 전에 "*args 가 와야 한다.

kwargs 는 딕셔너리 형태의 argument 가 된다. 


내가 넣어준 I=2,incidentangle=np.pi/2 는 딕셔너리의 component 들이 되는 것이다.


내가 key 와 value 를 지정해주면, 함수는 원래 기본값을 버리고 내가 넣어준 값으로 계산한다.


args 는 인자의 개수를 임의로 줄 수 있지만, 내가 준 인자가 무엇인지 바로 파악하기 힘들어보인다.


지금 같이 각 물리량들을 지정해주고 싶을 때에는 kwargs 를 쓰는 것이 더 나아보인다.


기본적으로 입사하는 E field 의 크기는 1 이고 입사각은 0 도 이지만, kwargs  를 이용해 원하는 물리량만 원하는 값으로 그때그때


바꿔서 함수를 사용할 수 있도록 만든 것이다.

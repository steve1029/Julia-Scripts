gcc 로 컴파일 하기 밑 shared object 만들기

리눅스/리눅스 일반

2017-08-04 11:21:46

% 컴파일 하기

$ gcc [소스코드] -o [출력파일명]

% shared object 만들기
.so 파일을 만들기 위해서는 일단 -fPIC 옵션(position independent code)으로 컴파일 하여 object 파일을 만들고 .so 를 만들어야 한다.
$ gcc -c -fPIC [소스코드] -o [출력오브젝트파일명]

$ gcc [입력오브젝트파일명] -shared -o [출력so파일명]


한방에 하고 싶으면


$ gcc -shared -o [출력so파일명] -fPIC [소스코드명]

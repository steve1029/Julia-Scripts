일단 윈도우에서 공식 홈페이지로 들어간 다음 blas 최신 버전 압축파일을 받는다.


혹은 리눅스에서 바로 다운 받는 방법도 있다.

get http://www.netlib.org/blas/blas.tgz



그리고 리눅스로 옮기고 압축을 풀어 준다.


# tar -xvf blas-3.7.0.tgz


그리고 압축 푼 폴더로 들어간 뒤 다음을 실행한다.


# cd ./BLAS-3.7.0



gunzip blas.tgz

tar xf blas.tar

cd BLAS

f77 -c -O3 *.f   # compile all of the .f files to produce .o files

ar rv libblas.a *.o    #  combine the .o files into a library

su -c "cp libblas.a /usr/local/lib"   # switch to root and install


여기서 f77 은 포트란 컴파일 명령어이므로 command not found 에러가 뜨면 aptitude 에서 다음을 설치하자.


# aptitude install -y fort77

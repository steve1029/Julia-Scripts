unsupported locale setting 에러가 날 때.

리눅스/시행 착오 노트

2017-07-10 10:39:48

서버에서 위 에러가 날 땐 다음의 command 를 입력하자.

# export LC_ALL="en_US.UTF-8"
# export LC_CTYPE="en_US.UTF-8"
# sudo dpkg-reconfigure locales

chroot 로 노드의 루트파일 시스템을 쓸 떄 위 에러가 난다면 다음을 입력하자.
# export LC_ALL=C


하지만 그래도 안될 시에는 다음 블로그에 따라 노드의 루트파일 시스템에 다음의 명령을 입력한다.

https://blog.seotory.com/solution/2017/02/docker-container-locate-error-fix


# localedef -v -c -i en_US -f UTF-8 en_US.UTF-8


그 다음 언어팩이 설치 됐는지 확인해보자.


# locale -a

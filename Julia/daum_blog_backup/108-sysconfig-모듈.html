자세한 건 공식홈페이지를 참조.


https://docs.python.org/2/library/sysconfig.html#module-sysconfig


파이썬 2.7 에서 새로 추가된 모듈이다. Disutils 패키지에 있던 모듈인데, 그 성능이 좋아서 기본 모듈로 아예 포함되버렸다.


파이썬의 system 파일 정보를 출력하고 볼 수 있다. 가령 설치 경로나 파이썬 버전, 설정 변수, 현재 플랫폼 등을 알 수 있다.


주요 소스코드는 여기서 참조하자.


https://hg.python.org/cpython/file/2.7/Lib/sysconfig.py


>>>sysconfig._main()


을 입력하면 sysconfig 에서 다루는 모든 변수들이 출력된다. 이 또한 소스코드에서 확인해볼 수 있다.


소스코드 맨 첫줄에 보면


import os

import sys

from os.path import pardir, realpath


등이 있는 것으로 보아, sys와 os 모듈을 잘 알아야 함을 알 수 있다.


참고로 sys.version 을 출력하면 파이썬 버전을 확인가능하다. 이 기능을 이용했음을 sysconfig 의 소스코드에서 확인 가능.

메모리 대역폭

컴퓨터 기본 및 상용 프로그램

2016-09-22 13:52:25

영어로 Memory Bandwidth 라고 하며

메모리와 다른 장치들 사이에서 한번에 전송될 수 있는 데이터의 양을 말함. 즉 단위 가 byte/sec 임.


보통 GB/sec 가 쓰인다.


전송되는 양은 버스 * 클럭 으로 계산한다.


버스는 한 번에 전송 되는 데이터의 양이고 클럭은 초당 전송 되는 횟수이다.


현재 DDR3 메모리를 기준으로 PC3-12800 이라고 되어 있는데 12800이 대역폭을 의미한다. 


다나와 제품 설명서에 보면 동작 클럭이 1600 MHz 라고 표기되어 있는데,


DDR3 제품은 실제 동작클럭이 200 MHz/s 이며 한 주기당 2^3 번 데이터를 전송하여 1600MHz/s = 1.6 GHz/s 라는 계산이 나온다.


12800 은 12.8 Gbyte/s를 의미한다. 


1.6 GHz(클럭, 주파수 단위) * 8 byte = 12.8 Gbyte/s


1초당 64bit 씩 10번 전송되면 전송되는 양은 640bit 라 할 수 있다.


제일 이상적인 메모리 버스는 CPU의 외부 버스, 그래픽 카드 버스 등을 모두 아우르는 크기이다. 즉 CPU 버스가 10bit 이고 


그래픽 카드 버스가 10bit 라면, 메모리 버스는 20 bit가 되어야 메모리와 cpu 사이, 그래픽카드와 메모리 사이의 데이터 전송이


서로 방해 받지 않을것이다. 그러나 실제로 그렇게 메모리를 구성하는 것은 매우 돈 낭비. 동시에 모든 부품이 최대 버스를 


사용하는 때는 거의 없기 때문이다. 그래서 메모리 버스의 크기는 cpu 외부 버스 보다 약간 큰 정도이면 충분하다.

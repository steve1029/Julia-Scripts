/etc/fstab 은 어떤 설정 파일?

리눅스/리눅스 일반

2016-09-29 23:36:45

# vi /etc/fstab

을 한 번 해보자. 여기엔 리눅스가 부팅 되면서 자동으로 마운트 시켜야할 장치들이 기록되어 있다.

만약 usb 같은 것을 리눅스에 연결한다면, 리눅스는 윈도우와 달리 불친절하여 연결하기만 하면 자동으로 인식되어 사용할 수 있는 것이 아니라 직접 마운트를 시켜줘야 한다.

예를 들어 usb 를 연결했더니 /dev/sdc 라고 인식 되었다고 하자. 이건 cfdisk 나 parted 에서 확인할 수 잇을 것이다. 이제 이 것을 수동으로 마운트 시켜줘야 한다.

보통 이런 device 들의 마운트 포인트는 /mnt 이다. 여기에 새로운 폴더를 하나 만들자

# mkdir /mnt/usb

원래 텅 비어 있는 /mnt 폴더에 usb 라는 폴더를 만들어 주었다.
# mount /dev/sdc /mnt/usb

이러면 이제 /mnt/usb 폴더로 들어가면 내가 꽂아준 usb 의 내용물이 보일 것이다.
'
하지만 이 짓을 매번 하기 귀찮고, 리눅스가 부팅 될 때 자동으로 마운트 시키도록 하고 싶다면 /etc/fstab 에 기록하기만 하면 된다!

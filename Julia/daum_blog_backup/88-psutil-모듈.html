파이썬을 이용해서 시스템의 정보 및 실행 중인 프로세스들의 정보를 파악할 수 있는 함수를 내장하고 있는 패키지이다.

리눅스, 윈도우, 맥OS 모두 가리지 않고 psutil 코드를 이용해 cpu 정보, 메모리 정보, gpu 정보등을 알아낼 수 있다. 역시 파이썬은 편리해.

심지어 Sun solaris, FreeBSD, OpenBSD 같은 마이너한 운영체제에서도 쓸 수 있는 강력한 패키지이다. 32bit, 64bit 운영체제 모두에서 작동한다.

파이썬 버전 2.6부터 추가된 패키지이며, 아나콘다 파이썬에는 기본으로 내장되어 있다. 물론 파이썬 3에는 당연히 포함되어있다.

기본적인 함수들을 살펴보자.

import psutil as ps

ps.cpu_times(percpu=True)

이렇게 하면 cpu time 에 대한 정보를 준다. cpu time 이 뭔지는 아직 모르겠다. 하여간 코어가 4개인 cpu를 사용한다면,

각 cpu에 대한 정보를 줄 것이다.

ps.cpu_time()

이렇게만 하면 맨처음 코어에 대해서만 정보를 출력한다.

ps.cpu_count()

이건 코어의 개수를 출력하는 함수이다.

ps.cpu_stats()

이런 함수도 있다.



메모리에 대한 정보를 출력할 수 있는 함수들은 다음과 같다.

ps.swap_memory()

ps.virtual_memory()

이런거 등등...

디스크에 대한 정보를 출력할 수도 있다.

psutil.disk_partitions() : 각 디스크의 파티션과 마운트 포인트, 파일시스템 타입, option 등을 출력한다. \
이 옵션은 읽기쓰기 가능여부, 제거가능 여부등이 포함되어 있다.

psutil.disk_usage('/') : 각 디스크의 용량 현황을 출력한다.

psutil.disk_io_counters(perdisk=False) : 이건 해당 디스크가 몇 번 쓰여졌고 얼마나 읽혔고 하는 등등의 정보를 출력한다.

이외에도 다양한 시스템 정보들을 출력할 수 있는데, 몇가지 재밌는 함수로는

ps.boot_time() : 이건 부팅시간을 리턴한다.

psutil.users() : 현재 유저 정보를 리턴한다.


또한 현재 시스템에서 실행되고 있는 프로세스들의 정보도 불러올 수 있다.

ps.pids() : 현재 실행되고 있는 프로세스들의 아이디 목록을 불러온다.

p = ps.Process(7055) : 현재 프로세스들 중에 pid가 7055 인 애만 하나 뽑아온 것.

p.name() : 이 프로세스의 이름.

p.exe() : 실행파일의 위치?

p.cwd() : 실행 명령을 내린 위치?

p.status : 현재 실행중인지 아닌지를 알 수 있다.

p.username() : 실행시킨 사람의 아이디. 서버 관리자는 서버에서 실행되는 프로세스를 확인할 때 누가 실행시켰는지 알아야할 때가 있겠지.

p.create_time() : 실행된 시각.

p.cpu_times() : 뭔지 모르겠다.

p.cpu_percent(interval=1.0) : 이 프로세스가 차지하는 cpu 비율을 1초 간격으로 측정한다.

p.cpu_affinity() : 이 프로세스가 접근할 수 있는 cpu의 코어들을 나열한다.


뭐 등등 여러가지 함수가 있다. 자세한 정보는 https://pypi.python.org/pypi/psutil 에서 확인하자.

Documentation 주소는 https://pythonhosted.org/psutil/ 이다.

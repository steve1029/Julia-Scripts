yhome3의 openmpi 버전이 노드의 openmpi 버전과 다르게 되면서 코드가 돌아가지 않게 됨.

gpu 노드에 mpi4py 와 openmpi 를 설치하니 자동으로 dependency 가 있는 앱들이 업뎃이 됨. 그러면서 yboot 의 nfs-kernel-server 앱이

새로 업뎃이 필요하게 되었음. 하지만 그게 자동으로 업뎃이 안됨. 원격 부팅이 불가해짐. nfs-server-kernel 을 업뎃하면서 문제가 해결 됨.

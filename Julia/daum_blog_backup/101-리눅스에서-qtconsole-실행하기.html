ssh 로 리눅스 원격 접속을 한 뒤에, qtconsole을 사용하고 싶다고 하자.


먼저 리눅스 컴에 ipython-qtconsole 과 python-sympy 를 설치한다.


# aptitude install -y ipython-qtconsole python-sympy


그 다음 윈도우 컴에서 Xming 을 켜고, xshell 에서 x11 포워딩을 x display 로 바꿔준다.


그리고 리눅스 컴에서 다음의 파일을 수정해줘야한다.


# vi ~/.bashrc


마지막 줄에 다음을 추가하자.


export $(dbus-launch)


이제 ssh 재접속을 한뒤 리눅스에서 실행하자


# ipython qtconsole

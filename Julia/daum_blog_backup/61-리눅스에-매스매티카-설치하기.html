리눅스에 매스매티카 설치하기

컴퓨터 기본 및 상용 프로그램/Mathematica

2016-09-30 10:18:44

리눅스용 매스매티카 설치 파일은 bash 파일이다.


# bash ./매스매티카.sh


설치가 바로 시작된다. 설치할 폴더와 실행파일의 위치를 입력하는 과정이 나올 것. 모두 기본 디렉토리로 하는 것이 제일 좋지만,


나는 일단 설치 디렉토리는 /mathematica 로 바꾸었다. 실행 파일의 위치는 반드시 bin 폴더에 들어가야한다는 걸 잊지 말자.



만약 설치가 실패하면서 인스톨에러가 나오고,


# vi 매스매티카설치폴더/InstallError


이 스크립트에서 다음과 같은 메세지가 뜬다면 


xdg-desktop-menu no writable system menu directory found


다음 명령을 수행하고 다시 설치하면 잘 될 것이다.


# mkdir /usr/share/desktop-directories/

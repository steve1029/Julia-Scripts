Nvidia cuda 를 VS 에서 사용하기

컴퓨터 기본 및 상용 프로그램/Visual Studio

2018-01-23 10:44:49

윈도우에 nvidia driver 를 설치하고 cuda 도 돌릴 예정이라면,


먼저 VS 를 깔기전에 게임한다고 설치한 nvidia driver 가 있는지 살펴보자. 그거 있으면 안된다 지우고


VS 설치한 다음에 nvidia toolkit 을 다운받아서 다시 설치해야한다.


먼저 Nvidia Developer zone 에서 Documentation 을 참고하자.


http://docs.nvidia.com/cuda/cuda-installation-guide-microsoft-windows/index.html


여기에서 현재 Nvidia 에서 배포하고 잇는 cuda toolkit 의 버전과 호환되는 VS 의 버전을 확인할 수 있다.


18.01.23) 현재 CUDA Driver 버전은 9.1 이며, VS 2017 을 지원한다.


1. VS 설치

현재 VS community 의 버전을 호환하는지 확인했다면, VS 를 다운 받아 설치한다.


2. Nvidia cuda toolkit 다운 및 설치

nvidia 홈페이지에서 다운 받아서 설치 프로그램을 실행한다. 이 때 반드시 기본 설치 말고 사용자 정의 설치를 누른다. 목록에서 VS 통합패키지가 체크되어 있는지 확인한다.

설치될 때 같이 딸려오는 subpackage 들은 위 nvidia doc 사이트에서 확인할 수 있다.


3. CUDA samples 실행

sample 코드들을 실행한다. 


만약 CUDA 10.0.props 가 없다는 에러가 뜨면서 프로젝트 파일이 VS 에서 제대로 로드되지 않는다면, CUDA Toolkit 을 설치할 때 VS 통합 패키지가 제대로 설치되지 않은 것이다. 제어판에서 NVIDA CUDA Visual Studio Integration 을 찾아 지우고, 다시 설치한다.


솔루션이 제대로 로드되었으면 빌드 해보자. 만약 'window sdk <버전>' 이 없다는 에러가 난다면 VS installer 를 시작프로그램에서 실행하여 해당 버전을 찾아 설치하면 된다.

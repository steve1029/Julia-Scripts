time 모듈

컴퓨터 언어/Python

2016-10-07 14:05:47

1. time.time

1970년 1월 1일 0시 0분 0초를 기준으로 현재까지 지난 시간을 '초'로 환산해준다.

>>> import time

>>> time.time()

2. time.localtime

>>>time.localtime(time.time())

time.time() 으로 반환된 초 단위 시간을 연도, 월, 일, 시간, 분, 초 로 바꾸어서 튜플로 나타내준다.

3. time.asctime

위 time.localtime 에 의해 반환된 튜플을 알아보기 쉬운 string 으로 출력해준다.

>>> time.asctime(time.localtime(time.time()))

4. time.ctime

위 time.asctime(time.localtime(time.time())) 를 간편하게 쓰기 위해 만든 함수. asctime과 다른 점은

ctime은 current time, 즉 현재 시간만을 반환하는 함수이다.

>>> time.ctime()

5. time.strftime

시간에 관계된 것을 세밀하게 표현할 수 있다.

time.strftime('출력할 형식 포맷 코드', time.localtime(time.time())) 꼴로 쓰면 된다.

시간에 관계된 것을 표현하는 포맷 코드

포맷코드 설명 예

%a 요일 줄임말 Mon

%A 요일 Monday

%b 달 줄임말 Jan

%B 달 January

%c 날짜와 시간을 출력함 06/01/01 17:22:21

%d 날(day) [00,31]

%H 시간(hour)-24시간 출력 형태 [00,23]

%I 시간(hour)-12시간 출력 형태 [01,12]

%j 1년 중 누적 날짜 [001,366]

%m 달 [01,12]

%M 분 [01,59]

%p AM or PM AM

%S 초 [00,61]

%U 1년 중 누적 주-일요일을 시작으로 [00,53]

%w 숫자로 된 요일 [0(일요일),6]

%W 1년 중 누적 주-월요일을 시작으로 [00,53]

%x 현재 설정된 로케일에 기반한 날짜 출력 06/01/01

%X 현재 설정된 로케일에 기반한 시간 출력 17:22:21

%Y 년도 출력 2001

%Z 시간대 출력 대한민국 표준시

%% 문자 %

%y 세기부분을 제외한 년도 출력 01

다음은 time.strftime을 사용하는 예이다.

>>> import time
>>> time.strftime('%x', time.localtime(time.time()))
'05/01/01'
>>> time.strftime('%c', time.localtime(time.time()))
'05/01/01 17:22:21'

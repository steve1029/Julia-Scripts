GRUB 이란?

컴퓨터 기본 및 상용 프로그램

2017-08-02 18:26:35

여러 개의 os 가 하드에 설치되어 있을 떄, 어느 것을 부팅할 지 알려주는 프로그램이다.

예를 들어 하드에 리눅스 하나만 설치되어 있다고 하자.

MBR 에 하드의 파티션 정보와 boot flag 의 위치가 기록된다.

boot flag 를 해당 하드의 1번 파티션에 설치했다고 하자.

컴퓨터가 부팅되면서 MBR 을 찾고, 거기서 boot flag 의 위치를 알아내어 boot flag 를 찾아낸다.

boot flag 에는 kernel 과 kernel module 가 어딨는지 기록되어 있다. Boot flag 를 읽음으로서 리눅스가 부팅된다.

하지만 os 가 여러개 설치되어 있다면 MBR 과 boot flag 가 여러 개일 것이고, 메인보드는 이 중에 무엇을 읽어야 할지

알수 없게 된다. 그 때 grub 이 무슨 os 를 부팅할지 알려주는 것.

따라서 요즘 os 는 grub 이 무조건 설치되며, grub 이 제일 먼저 실행된다. os 가 하나라면 grub 이 실행되도

아무런 일을 하지 않을 것이고, 여러 개의 os 가 설치되어 있다면 grub 이 실행되어

os 를 선택하라고 말할 것이다.

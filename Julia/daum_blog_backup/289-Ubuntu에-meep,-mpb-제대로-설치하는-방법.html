리눅스/리눅스 일반

2021-04-26 17:54:01

먼저 우선적으로 설치되어야 할 앱들.

Lapack, BLAS, FFTW3, gfortran, MPB, Libctl, Harminv, libGDSII, MPI, OpenMP, HDF5, Python, Guile.

 

대부분은 .tar.gz 파일을 공식홈페이지에서 다운 받은 다음, configure && make && make install 방법으로 설치하면 된다.

apt install liblapack-dev libblas-dev gfortran libfftw3-dev libfftw3-doc libffi-dev libgc-dev

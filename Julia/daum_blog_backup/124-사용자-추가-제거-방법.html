사용자 추가


# adduser 사용자이름

$ vi /root/etc/group

여기엔 각 사용자들의 id 와 그에 해당하는 uid(userid)가 적혀있다.

uid는 리눅스에서 실제로 인식하는 id 로, 우리가 사용하는 id는 그냥 보여지는 것일뿐 시스템은 이 숫자로 유저를 인식한다.

1000번 이전의 uid 는 시스템 uid 이므로 건드리지 말것. 사용자들은 보통 1000번 대 이상의 uid를 부여받는다.

나머지 gshadow, shadow, passwd 에도 유저 정보가 들어 있다. 비밀번호는 passwd 파일에 있는데, 암호화가 되어 있어

열어봐야 알아볼 수는 없다.

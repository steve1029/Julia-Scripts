기본적인 사용법은 여기 나온 대로 따라서 하면 된다. sublime text 3 에서도 먹히는 사용법임.


sublime text 는 윈도우에서 사용하는 에디터. 윈도우에서는 latex 용 에디터로는 유명한 것이 textmaker, texworks 등이 있다.


하지만 난 가벼운 것을 선호하므로 서브라임 텍스트를 쓸 것이다.


온라인에서 latex 문서를 관리 하는 방법도 있다. sharelatex 라고 하는 사이트를 가면 된다. 하지만 나는 일단 내 클라우드에서 작업할 것.


윈도우에서 쓸 latex 컴파일러는 tex live, miktek 가 있는데, 나는 miktek 을 쓸 것이다.


miktex 을 설치하면 texworks 가 자동으로 같이 설치된다는 것을 참고하자.


그리고 sumatrapdf 를 설치한다. 무료 pdf 뷰어이고, sublimetext 에서 사용하는 pdf 뷰어이다. 매우 가볍고 좋은 프로그램.


sumatraPDF 는 설치할 때 환경변수에 자동 등록이 안되기 때문에 등록을 꼭 시켜줘야 서브라임 텍스트가 컴파일 하고 난 뒤


sumatraPDF 를 찾을 수 있다. 환경변수의 시스템변수의 Path 를 편집하여 sumatrapdf 폴더의 경로를 추가하자.


서브라임텍스트 3(2) 를 설치하고


서브라임텍스트 패키지 컨트롤을 설치한다. 이건 서브라임 텍스트의 패키지를 관리할 수 있는 툴인 것 같다.


그리고 이 툴 관리 앱을 이용해 LaTexTools 라는 서브라임텍스트 패키지를 설치해야 된다.


이러면 끝. 이제 예제 파일을 ctrl + b 로 컴파일 해보자.

우분투 업그레이드 방법 및 upgrade / dist-upgrade / do-release-upgrade 차이

리눅스/우분투 서버 구축

2020-08-12 18:25:14

우분투를 업그레이드 하는 방법은 3가지가 있다

 

upgrade / dist-upgrade / do-release-upgrade

 

여기서 현재 코드네임을 유지하면서 (예를들어 18.04의 코드네임은 bionic beaver 이다.)

 

패키지만 최신으로 업그레이드 하고 싶다면

 

$ sudo apt update 
$ sudo apt upgrade
$ sudo apt dist-upgrade

 

를 수행한다.

 

나는 이 방법을 매우 선호하지만, 릴리즈 업그레이드까지 해야한다면 다음을 하자.

 

$ sudo apt install update-manager-core
$ sudo do-release-upgrade

 

만약 No new release found 라는 에러 메세지가 나온다면 다음을 실행하자.

 

$ sudo do-release-upgrade -d

 

위 명령어를 실행하면 업그레이드가 시작되는데, 무슨 무슨 메세지가 여러번 나온다. 전부 y 를 하면 됨.

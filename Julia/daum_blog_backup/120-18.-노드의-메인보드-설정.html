18. 노드의 메인보드 설정

리눅스/데비안 서버 구축

2017-01-10 16:45:55

네트워크 부팅을 하는 노드에서는 메인보드의 특정 기능을 꺼주어야만 할 때가 있다. 그 기능들은 다음과 같다.

wait for key if error : 로딩 과정에서 에러 나면 키를 입력해야 다음으로 넘어감. 키보드를 연결하지 않을 것이므로 체크 해제.

Highper Threading : FDTD 계산할 때 이거 켜 놓으면 느려진다.

fast boot : 이 전 부팅 기록을 다시 읽어서 부팅을 빠르게 하는거임. 체크해제.


메인보드에 따라서 전원이 강제로 off 됐을 때 메인보드 설정이 초기화 되는 놈들이 있다. ex) 기가바이트


따라서 이런 보드에서는


전원 off 시 시스템 상태 : 이전 상태로 돌아가기


를 반드시 설정해주어야 한다.

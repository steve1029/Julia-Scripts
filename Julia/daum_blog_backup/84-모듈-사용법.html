파이썬은 기본적으로 많은 모듈을 가지고 있다.

모듈을 import 하면 파이썬은 어떻게 그 모듈을 불러오는가?

1. 현재 Script가 포함된 디렉토리

2. 환경변수 PYTHONPATH 에 지정된 경로

3. Python이 설치된 경로 및 그 밑의 라이브러리 경로

이 순서대로 모듈을 찾아서 불러온다.

더 정확히 알고 싶다면 ipython을 열고

In[1]:import sys

In[2]:sys.path

라고 친다면 모듈을 불러오는 다양한 경로들을 확인 할 수 있다.


용어 설명

1. 패키지란?

패키지는 많은 모듈들을 모아놓은 폴더이다.

가령 numpy scipy matplotlib은 모두 패키지들이며, 그 안에 많은 모듈이 들어 있다.

패키지는 기본적으로 폴더이며, 모듈은 다 파이썬 스크립트이다.

'모듈을 불러온다' 라고 함은 다른 파이썬 스크립트의 내용을 불러와서 쓰겠다는 뜻이다.

다시 말해 '모듈로써 불러온다' 라고 해야 정확한 표현이다.

import scipy.constants

라고 하면 scipy 패키지 안의 constants 폴더의 모든 내용을 import 하겠다는 뜻이다.

먼저 파이썬 인터프리터는 위 세가지 경로를 통해 scipy 라는 폴더를 찾을 것이며

그 안의 constants 폴더를 찾아서 그 안의 모든 스크립트 파일들을 불러올것이다.


2. __init__.py

그 폴더를 모듈로서 import 할 때, 파이썬 인터프리터는 __int__.py의 내용을 자동으로 실행한다.

어떤 스크립트가 모듈로서 import 될 수 있도록 설정하려면 반드시 그 폴더 내에 __init__.py가

있어야 한다. 또한 그 폴더가 모듈로 import 될 때 뭔가가 자동적으로 실행 되길 원한다면

__init__.py 내용 안에 몇가지를 추가해주면 된다. 가령 matplotlib 은 import 될 때 numpy를 자동으로

import 하도록 되어 있다.

import matplotlib

이 내용이 numpy의 __init__.py 에 들어 있을 것임을 알 수 있다.

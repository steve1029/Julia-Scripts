ls 명령어

2016-09-26 10:21:36

폴더의 내용물을 출력해주는 리눅스 기본 명령어. 윈도우에서의 dir 과 같다.

ls sd* : 인식한 저장장치(하드나 ssd 등)과 마운트된 파티션들을 보여준다.

ls -a : all. 숨겨진 파일까지 전부다 보여줌

ls -l : 내용물을 자세하게 보여줌. 디렉토리인지 파일인지, 권한은 어떻게 되는지, 만들어진 날짜와 크기는 어떤지도 다 나온다.

ls -al : 숨겨진 파일까지 모두 다 자세히 보여줌. 즉 -a 와 -l 옵션을 합쳐서 나타낸 것.

'ls --help' : 간략한 설명을 보여줌

man ls : manual ls 라는 뜻으로 해당 명령의 자세한 설명을 보여줌.

ls -alph : a,l,p,h 옵션을 모두 적용해서 보여줌.

ls -lh : 폴더 내의 내용물을 크기과 함께 표기해서 보여줌. -h 옵션은 반드시 -l 이나 -s 와 같이 쓰여야 함.

ls -R : 서브 디렉토리의 내용물도 같이 보여줌.

ls -p : 디렉토리인지 파일인지도 보여주는 옵션. 디렉토리는 파일명 뒤에 / 가 붙는다.

Nvidia driver 와 cuda 를 설치한 후

리눅스/우분투 cuda 환경 구축

2018-12-15 15:14:41

두 개를 설치하고 난 뒤 nvcc command 를 쓰면 nvcc 가 설치 안되어 있다면서


apt-get install nvidia-cuda-toolkit

이걸 설치하라고 한다. 절대 하지 않는다.
이 것은 apt repository 에 올라와 있는 아주 예전버전의 nvidia toolkit 이다

이제 이 모든 것은 cuda 로 통일되었으므로 cuda 를 설치하면 nvcc 는 자동 설치된다.

설치되어 있는데 못찾는 것일 뿐이므로, 다음 명령어를 실행하면 된다.$ vi ~/.bachrc여기에 다음을 추가한다.

export PATH="$PATH:/usr/local/cuda-10.0/bin"

버전은 알아서 바꾸시길. 파일 닫고 다음 명령 실행.# source ~/.bashrc
이제 nvcc 를 찾을 수 있게 되었다.모든 사용자에 대해 추가하고 싶다면, /etc/bash.bashrc 에 해당 스크립트를 추가하고 source 하면 됨.

vim 단축키 정리

리눅스/vi 사용법

2016-09-26 09:49:46

입력


i : 커서 위치에 삽입


a : 커서 오른쪽 위치부터 삽입


o : 커서 다음 줄부터 삽입


A : 해당 줄의 맨 오른쪽에 삽입  


I : 해당 줄의 맨 왼쪽에 삽입


O : 커서 위에 삽입


r : 해당 위치를 다른 문자로 대체


R : 해당 줄을 다른 텍스트로 대체


cw : 단어 번경


cc : 행을 변경


r + enter : 문자열 중간에 빈 줄을 하나 삽입함.

삭제


x : 입력 모드에서 백스페이스의 기능을 함.


del : 명령모드에서 백스페이스의 기능을 함.


dw : delete word. 즉 단어 하나를 잘라내기. 단어는 보통 띄어쓰기 단위로 구별한다.


d$ : $는 문장 끝까지 가라는 명령어이다. 즉 커서위치부터 문장 끝까지 지운다. 혹은 D 를 누른다.


dd : 해당 문장 전체를 잘라내기

ex) 10dd 라고 하면 커서가 있는 줄을 포함하여 아래로 10줄을 지운다.


D : 커서 위치에서 행의 끝까지 삭제


취소


u : 명령모드에서 control + z 의 기능을 한다.


이동


w : 단어의 첫글자로 이동


W : 화이트 스페이스 단위로 다음 글자로 이동하기


b : 백워드 방향으로 단어의 첫글자로 이동하기


B : 백워드 방향으로 화이트 스페이스 단위로 다음 글자로 이동하기


^ : 줄의 맨 처음으로 이동


$ : 줄의 맨 뒤로 이동


H : 화면의 맨 위로 이동


M : 화면의 중앙으로 이동


L : 화면의 맨 아래로 이동


ctrl + f : 한 화면 아래로 이동 (forward)


ctrl + b : 한 화면 위로 이동 (backward)


ctrl + u : 페이지 절반만큼 다음으로 이동


ctrl + d : 페이지 절반만큼 이전으로 이동


ctrl + e : 한 줄 씩 아래로 이동


ctrl + y : 한 줄 씩 위로 이동


w,e,b : 다음 단어의 앞으로 이동, 다음 단어의 끝으로 이동, 이 전 단어의 앞으로 이동.


ctrl +g : 파일 이름과 현재 라인 번호, 총 라인 번호 등을 보여줌.


G : 현재 파일 마지막 행으로 감


ex) 12G : 현재 파일의 12번째 줄로 감.


% : 커서를 괄호 내부에 두고 사용하면 그 괄호의 짝으로 이동.


gg : 문서의 시작으로 이동


ge : 백워드 방향으로 단어의 마지막 글자로 이동하기.


f : 문자의 위치로 이동. f 를 누르면 계속 이동


F : 백워드로 문자의 위치로 이동.


명령어


set nu : 라인 번호 표시


set nonu : 라인 번호 표시한거 지움.



에러 수정


r,cw,cc 등을 살펴 보았었음.


:s/old/new/g


: 커서가 있는 줄의 모든 old 를 new로 바꾼다.


:%s/old/new/g : 파일 전체의 old를 new로 바꾼다.


:14,17s/old/new/g : 14~17 줄의 모든 old를 new로 바꾼다.


복사하기

명령어 모드

yy : 커서가 있는 줄을 통째로 복사한다.


비주얼 모드


비주얼 모드는 명령모드에서 v 키를 누르면 들어갈 수 있다.


이때 방향키를 움직이면 블록이 씌워진다.


y : yank 의 약자로 잡아당긴다는 뜻이다. 블록한 부분을 복사함.


p : paste. 붙여넣기.


비주얼 블록 모드


column 을 블록 씌울 수 있는 모드. ctrl+v 로 진입. gVim 에서는 ctrl+q 로도 진입가능.



붙여넣기


p : 커서가 있는 줄의 다음 줄에 잘라내기한 줄을 붙여 넣는다.

2p : 두 번 붙여 넣기


되돌리기


u : 윈도우에서의 ctrl+z 와 같다


되살리기


Ctrl +r : 윈도우에서의 ctrl + x 와 같다.


문자열 탐색

/ : ex) /plt 라 하면 plt가 들어간 글자를 정방향으로 찾음


? : 역방향으로 찾음


n : 다음에 매칭되는 단어로 가기


N : 이전에 매칭되는 단어로 가기



색깔 보이기


/home/ldg 에 다음과 같은 설정파일을 하나 만든다.


즉 /home의 유저 폴더 안에 만들면 그 유저는 그 설정파일을 쓰는것이다!


(전체 계정에 테마를 적용시키고 싶으면 /etc/vimrc 에 코드를 삽입한다)


vi ~/.vimrc


다음의 내용을 입력한다.


colo desert


syntax on


이제 이 홈디렉토리와 그 하위 폴더의 모든 vi 파일들은 이 테마의 적용을 받게 된다.


테마 종류는 많다.


blue, darkblue, default, delek, desert, elflord, evening, koehler, morning, murphy,


pablo, peachpuff, ron, shine, slate, torte, zellner


여기서 syntax 부분을 찾아 on 으로 바꿔준다.


설치되어 있는 vim 테마를 보려면 다음으로 이동한다.


$ cd /usr/share/vim/vim74/colors


이 안의 파일들이 vim 테마 파일들이다. 여기서 맘에드는 것을 선택해서 ~/.vimrc 에 입력해주면 된다.



줄 번호 보이기


vi 에디터에서 


:set nu


:set nonu


이러면 되지만, 매번 설정해줘야 하므로 기본 값으로 만드는 방법을 알아보자.


$ vi /home/ldg/.vimrc


여기에 다음 줄을 추가한다.


set nu


set smartindent


자동 줄번호, 자동 들여쓰기 기능을 이렇게 추가할 수 있다.


이 외 다른 설정들.


set number            " line 표시를 해줍니다.

set ai                    " auto indent

set si                    " smart indent

set cindent            " c style indent

set shiftwidth=4      " shift를 4칸으로 ( >, >>, <, << 등의 명령어)

set tabstop=4         " tab을 4칸으로

set ignorecase      " 검색시 대소문자 구별하지않음

set hlsearch         " 검색시 하이라이트(색상 강조)

set expandtab       " tab 대신 띄어쓰기로

set background=dark  " 검정배경을 사용할 때, (이 색상에 맞춰 문법 하이라이트 색상이 달라집니다.)

set nocompatible   " 방향키로 이동가능

set fileencodings=utf-8,euc-kr    " 파일인코딩 형식 지정

set bs=indent,eol,start    " backspace 키 사용 가능

set history=1000    " 명령어에 대한 히스토리를 1000개까지

set ruler              " 상태표시줄에 커서의 위치 표시

set nobackup      " 백업파일을 만들지 않음

set title               " 제목을 표시

set showmatch    " 매칭되는 괄호를 보여줌

set nowrap         " 자동 줄바꿈 하지 않음

set wmnu           " tab 자동완성시 가능한 목록을 보여줌

syntax on        " 문법 하이라이트 킴"

:vsplit

:sp

현재 열린 파일을 수평으로 2개 띄운다

ex) :vs newfile

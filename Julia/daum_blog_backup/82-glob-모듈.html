디렉토리 내의 파일들의 목록을 리스트로 만들어서 반환해 줄 수 있는 모듈이다. 조금 더 정확히 말하자면


unix shell 에서 쓰이는 파일 찾기 관련 방법들을 그대로 파이썬에서도 쓸 수 있게 해주는 모듈이다.


*.py 라고하면 unix 에서는 확장자가 .py 인 모든 파일을 의미한다. 이런 notation 들이 파이썬에서도 동작하도록


해주는 모듈이라 이 말이다.


공식홈페이지의 doc 를 참조하면, glob 내의 메쏘드는 두개 밖에 없다.

그 중 나는 아마도 glob.glob(pathname) 말고는 쓸 일이 없을 것 같다.


https://docs.python.org/2/library/glob.html

1. glob.glob

>>> import glob

>>> glob.glob("C:/Python/q*")

해당 디렉토리에서 q 로 시작하는 모든 파일들의 이름을 리스트로 만들어서 반환한다.

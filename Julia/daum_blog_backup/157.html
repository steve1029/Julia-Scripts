이제 각각에 대한 답을 달아 보겠습니다
=====================================================================
1. 기존에 /home 디렉토리를 사용중이나, 공간의 문제로 다른 파티션에 홈디렉토리를 생성하고 싶다.
1) useradd 했을때, 자동으로 자신의 원하는 위치(default로)로 생성되게 하고 싶다.

# man useradd <-- 전반적인 useradd의 과정을 알수 있습니다.
읽어 보시면 "/etc/default/useradd - default information" 부분을 보실수 있을 겁니다.

# vi /etc/default/useradd
# useradd defaults file
GROUP=100
HOME=/home
INACTIVE=-1
EXPIRE=
SHELL=/bin/bash
SKEL=/etc/skel

기본 정보를 확인 하실수 있을 겁니다. 여기서 HOME=/home을 변경해 주시면 초기 셋팅이 완료 됩니다.
** 관련 디렉토리는 꼭 생성해 주셔야 합니다. (HOME=/home1 이라 하시면 # mkdir /home1 을 꼭 만들어주셔야 합니다.)


2) 필요한 사용자만 따로 home 디렉토리를 지정해 주고 싶다.
위의 님 말씀 처럼
# useradd -m -d /home1/test111 test1111 <-- 마찬가지고 /home1은 만들어 주셔야 합니다.
기존 사용자를 변경 하시려면,
# usermod -d /home1/test1111 test1111 <-- 기존 사용자의 home 디렉토리 변경.
위 명령어를 간단하게 설명 드리면, test1111 사용자의 홈디렉토리를 /home1/test1111로 바꾸라는 말입니다.


2. /home이라는 디렉토리를 계속 사용하나 실질적으로는 다른 곳의 파티션을 사용하고 싶다.

위의 님 말씀 처럼 ln -s를 잡아 주시면 됩니다. 이 부분은 상세히 기재하지 않겠습니다.

MS 오피스 2010에서 Mathtype 사용하기
IT/Windows

Mathtype은 안타깝게도 아직 오피스 2010을 지원하지 않고 있지만,

다행스럽게도 약간의 작업만으로 오피스 2010 워드와 파워포인트에서 Mathtype을 사용할 수 있다.

이 글의 내용은 아래 출처에 있는 내용을 보기 쉽게 한글로 정리한 것이다.

< 출처: Mingliang's software development base >


1. MS Office Word 2010

먼저 Mathtype이 설치된 폴더에서 다음의 두 개 파일을 복사한다.

<Mathtype Installation Path>\MathPage\MathPage.wll
<Mathtype Installation Path>\Office Support\MathType Commands 6 For Word.dotm


그리고 이를 아래의 폴더에 붙여넣기 하면 된다.

<Office Installation Path>\Office14\STARTUP\


다시 워드를 실행하면 매크로 관련 경고가 뜨는데 '매크로 포함' 또는 '게시자의 모든 내용 신뢰'를 눌러 주면 완료된다.

2. MS Office PowerPoint 2010

파워포인트에서의 설정은 워드와는 조금 다르며, 아래의 내용을 따라하면 된다.

① 파워포인트 실행 후 좌측 상단의 ‘파일’을 클릭한 후, ‘옵션’을 클릭

② 좌측에 있는 항목 중 ‘추가 기능’을 클릭 한 후

③ 우측 하단 ‘관리’에 있는 ‘드롭-다운 리스트’을 눌러 ‘Power-Point 추가 기능’을 선택 ‘이동’ 클릭
 
④ ‘새로 설치’ 를 클릭하여 아래의 파일을 찾아 추가한다.

<Mathtype Installation Path>\Office Support\MathType AddIn (PowerPoint 2007).ppam


⑤ 매크로 관련 경고문 나오면 ‘게시자의 모든 내용 신뢰’ 또는 ‘매크로 포함’ 중 하나를 선택하면 설정 완료된다.

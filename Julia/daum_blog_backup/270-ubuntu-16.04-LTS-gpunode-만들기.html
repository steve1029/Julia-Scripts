ubuntu 16.04 LTS gpunode 만들기

리눅스/우분투 cuda 환경 구축

2019-03-13 17:18:10

1. ubuntu 를 설치한다.

2. 인터넷에 연결한다. 이때,랜카드 선택이 중요하다. 너무 최신 랜카드는 openmpi 1.10.2 버전에서 인식이 안될 수 있으니 무엇을 인터넷에 연결하고 무엇을 내부네트워크에 연결할지 잘 선택해야한다.

3. SSH 설정을 한다.


# apt install -y openssh-server sshpass


4. nvidia driver 를 설치한다.

5. pycuda 를 사용할 수 있도록 공식 홈페이지를 따라 설치한다.


# apt install nvidia-cuda-toolkit nvidia-cuda-dev python3-pycuda python3-numpy python-numpy


6. 서버의 사용자 정보를 노드에 복사한다.

7. 서버의 hosts 를 노드의 hosts 로 복사한다.

8. 서버의 dhcpd.conf 에 새노드에게 할당할 ip 를 설정한다. 그리고 /etc/init.d/isc-dhcp-server restart

9. 노드에 nfs-common 을 설치한다.

10. 노드의 /etc/fstab 에 다음 내용을 추가하여 서버의 /home 과 /root 를 마운트한다.


gpucluster:/root    /root   nfs defaults,async,nosuid,nodev,intr,nfsvers=3,rsize=32768,wsize=32768  0   0

gpucluster:/home    /home   nfs defaults,async,nosuid,nodev,intr,nfsvers=3,rsize=32768,wsize=32768  0   0

gpucluster:/home/ldg    /home/ldg   nfs defaults    0   0

gpucluster:/home/cris123    /home/cris123   nfs defaults    0   0


마지막 두 줄은 사실 없어도 된다.

11. mount -a 로 마운트하자.

12. 제대로 마운트 됐다면 test code 를 돌려보자.

13. mpi4py 를 설치한다.


# apt install -y openmpi-bin openssh-client openssh-server libopenmpi-dev

# apt install -y python-mpi4py python3-mpi4py


14. test code 를 돌려본다.

15. 그 외 나머지 파이썬 모듈 및 C 언어용 library 들을 설치한다.

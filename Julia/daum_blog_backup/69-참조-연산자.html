1. 역참조 연산자 '*'


포인터의 역참조 연산자 '*' 는, 주소값을 받아서 그 주소에 저장된 값을 가져오는 연산자이다. 즉 주소에 있는 데이터를 읽기 위한 연산자임. 반대로 포인터 주소에 데이터를 쓸 때에도 사용이 된다. 알 수 있겠지만 * 뒤에는 반드시 포인터가 와야한다.


int *address_of_data = &address;

int data = *address_of_data


첫줄에서 address_of_data 라는 포인터 변수를 선언. 즉 주소값을 저장하는 어떤 메모리 공간을 설정하고 ,address_of_data 라는 이름을 붙임.v  그리고 그 값에는 당연히 주소값을 넣어줘야겠지. 그 주소값은 address 라는 변수의 주소값이다. data 는 address_of_data 변수의 값을 저장하였다. 


즉 * 뒤에는 메모리주소값이 와야 한다.


2. 주소 연산자 '&'


데이터를 받아서 그 데이터가 저장된 주소를 가져온다. 즉 & 뒤에는 반드시 변수가 와야한다. 변수가 오는 것은 변수 안의 데이터가 오는 것과 같다.


가령 


int x;

라 할 때 &x 는 x 라는 변수(데이터)가 저장된 주소를 가져옴.


참고) 포인터 배열이란?


일련의 메모리 주소를 배열에 저장한 배열을 말한다. 즉 포인터를 저장한 배열이다.


정리)


변수의 주소는 메모리 어디에서 변수를 찾을 것인지 알려준다. 따라서 이런 변수의 주소를 포인터라 부른다. 포인터 변수는 메모리 주소를 저장하는 변수이다. 그리고 포인터 변수를 선언할 때에는, 그 포인터가 가르키는 주소에 어떤 종류의 데이터가 저장되어 있는지 알려주어야 한다.

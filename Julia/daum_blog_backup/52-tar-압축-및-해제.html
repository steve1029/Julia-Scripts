tar 압축 및 해제

리눅스/리눅스 일반

2016-09-28 11:52:21

1. tar로 압축하기
> tar -cvf [파일명.tar] [폴더명]

ex) abc라는 폴더를 aaa.tar로 압축하고자 한다면
> tar -cvf aaa.tar abc

2. tar 압축 풀기
> tar -xvf [파일명.tar]

ex) aaa.tar라는 tar파일 압축을 풀고자 한다면
> tar -xvf aaa.tar

3. tar.gz로 압축하기
> tar -zcvf [파일명.tar.gz] [폴더명]

ex) abc라는 폴더를 aaa.tar.gz로 압축하고자 한다면
> tar -zcvf aaa.tar.gz abc

4. tar.gz 압축 풀기
> tar -zxvf [파일명.tar.gz]

ex) aaa.tar.gz라는 tar.gz파일 압축을 풀고자 한다면
> tar -zxvf aaa.tar.gz


 옵션

 설명

 -c

 파일을 tar로 묶음

 -p

 파일 권한을 저장

 -v

 묶거나 파일을 풀 때 과정을 화면으로 출력

 -f

 파일 이름을 지정

 -C

 경로를 지정

 -x

 tar 압축을 풂

 -z

 gzip으로 압축하거나 해제함



권한까지 압축하기 및 풀기


압축 : # tar cvfzp filename.tar.gz filename/

filename 폴더 안의 모든 내용이 권한도 포함하여 filename.tar.gz 로 압축됨. 현재폴더에 새로운 디렉토리가 형성되면서.

풀기 : # tar xvfzp filename.tar.gz -C filename/


filename 폴더 내에 압축 내용을 품. 권한까지.

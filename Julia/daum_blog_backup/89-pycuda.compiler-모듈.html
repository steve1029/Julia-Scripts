from pycuda.compiler import SourceModule 이라고 하면


해당 코드를 컴파일 하는데에 SourceModule 이라는 방법을 이용하겠다는 뜻이다.


GPU에서 돌아갈 커널 함수를 pycuda 에서 이용하는 방법은 여러가지가 있는데 그 중에 하나는


이미 다른 스크립트 파일로 만들어진 커널 함수를 불러와서 쓰는 방법과,


해당 스크립트 내에 string 으로 커널 함수를 써주고, 그걸 알아서 모듈화하여 이용하도록 명령을 내리는 방법이 있다. SourceModule 은 그 중


후자에 해당한다.


mod = SourceModule("""

__global__ void doublify(float *a)

{

int idx = threadIdx.x + threadIdx.y*blockDim.x;

a[idx] *= 2;

}

""")


string 안의 내용들이 커널함수로 컴파일 되어 GPU 에서 작동할 것이다.


반면 외부의 cuda 스크립트에서 불러올 경우 pycuda.driver 의 Module 기능을 이용하면 된다.

윈도우/mac/리눅스에 git repository 연결하기

컴퓨터 기본 및 상용 프로그램/Github

2019-02-17 22:27:59

내 리눅스 코드들은 모두 git의 http://github.com/steve1029/MSdegree 에 있다.

이를 윈도우에 백업해두려면, 일단 윈도우용 git 을 설치하자.
그러면 git bash 가 설치된 것을 알 수 있다. 이를 실행한다.

그 다음 할 일은 간단하다.



git bash 는 윈도우에서 리눅스의 명령어로 git 을 다룰 수 있게 해주는 프로그램이다.
일단 테스트 폴더 하나 만들자

$ mkdir gittest
$ cd gittest
$ git init
$ git remote add origin https://github.com/steve1029/Msdegree : 가져오거나 보내는 곳을 설정 해주자.
$ git remote -v :가져오고 보내는 곳이 잘 설정됐는지 확인
$ git pull origin master : 가져오기!

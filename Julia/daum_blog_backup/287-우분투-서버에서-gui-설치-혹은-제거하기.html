우분투 서버에서 gui 설치 혹은 제거하기

리눅스/우분투 서버 구축

2020-09-15 11:15:12

우분투 서버에 gui를 설치하고 싶다면 다음을 입력하자.

sudo apt-get install ubuntu-desktop

모르고 gui를 설치해버린 다음에 지우고 싶다면 다음 명령어를 입력하자.

sudo apt-get -y purge ubuntu-desktop
sudo apt-get -y purge unity gnome-shell lightdm slim
sudo apt-get -y remove ubuntu-desktop
sudo apt purge ubuntu-desktop -y && sudo apt autoremove -y && sudo apt autoclean
sudo apt-get purge ubuntu-desktop -y
sudo apt-get -y clean
sudo apt-get -y autoremove
sudo apt-get -f install

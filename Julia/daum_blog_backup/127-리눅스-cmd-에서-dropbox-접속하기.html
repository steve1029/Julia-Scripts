리눅스 cmd 에서 dropbox 접속하기

리눅스/서버 앱 설정 방법

2017-01-18 14:35:11

먼저 curl 이라는 앱을 설치해야 한다.

# aptitude install -y curl

그 다음 dropbox uploader 를 다운 받는다.

# wget https://raw.github.com/andreafabrizi/Dropbox-Uploader/master/dropbox_uploader.sh
# chmod 755 dropbox_uploader.sh

이제 다운받은 것을 실행해본다.

./dropbox_uploader.sh

그러면 어쩌구 저꺼구 실행되면서 access token 을 입력하라고 뜬다.

그럼 이 access token 을 알아내기 위해 데스크톱에서 다음 사이트로 접속한다.

https://www.dropbox.com/developers/apps

여기서 새로운 dropbox 애플리케이션을 만들어야한다. 우측 상단의 Create app을 누른 뒤

Dropbox API 를 선택하고 access 는 Dull Dropbox 라고 한 뒤 앱의 이름도 설정하면 됨.

앱이 다 만들어지면 그 앱의 setting 탭에서 Generated access token 이라는 항목이 있다. Generate 을 누른 뒤

거기에 나온 내용을 아까 리눅스 쉘에 입력해주면 된다.

다시 ./dropbox_uploader.sh 를 실행하면 명령어들이 설명될 것.
